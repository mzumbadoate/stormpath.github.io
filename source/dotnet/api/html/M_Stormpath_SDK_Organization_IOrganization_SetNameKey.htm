nameKey
Type: SystemString
The Organization's name key. Organization name keys must be unique within a Stormpath Tenant, and must follow DNS hostname rules: it may only consist of a-z, A-Z, 0-9 and - (hyphen). It must not start or end with a hyphen. The uniqueness constraint is case-insensitive.
