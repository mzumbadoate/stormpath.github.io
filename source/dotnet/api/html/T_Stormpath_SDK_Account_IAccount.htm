IAccount Interface
An Account is a unique identity within a Directory. Accounts within a Directory or Group mapped to an Application may log in to that Application.

SetPassword
Sets (changes) the account's password to the specified raw (plaintext) password. Only call this method if you legitimately want to set the account password directly. When possible, it is advisable to use Stormpath's account password reset email workflow instead.

After calling this method, you must call SaveAsync(CancellationToken) to propagate the change to the Stormpath servers.

SetStatus
Sets the account's status. Accounts that are not Enabled may not login to applications.

SetUsername
