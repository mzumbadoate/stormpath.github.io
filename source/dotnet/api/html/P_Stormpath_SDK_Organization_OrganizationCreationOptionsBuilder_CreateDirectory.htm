If , a new Directory will be created. The new Directory will automatically be assigned as the Organization's default Account and Group store. If the DirectoryName property is not null, the new Directory will be created with that name. Otherwise, the Directory will be automatically named based on heuristics to ensure a guaranteed unique name based on the Organization.

If , no Directory will be created.
